Delphi project files

Although these are installation dependant they are provided for informaton.

In Project->Options->Delphi Compiler-All Configuration->Conditional Defines add an entry USEDLL.  If you do not
make this entry then the compiler will attempt to compile the DBIsam code and will fail.
